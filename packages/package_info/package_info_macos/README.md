# PackageInfo MacOS

MacOS implementation of package_info.

**Please set your constraint to `package_info_macos: '>=0.4.y+x <2.0.0'`**

## Backward compatible 1.0.0 version is coming
The package_info_macos plugin has reached a stable API, we guarantee that version `1.0.0` will be backward compatible with `0.4.y+z`.
Please use `package_info_macos: '>=0.4.y+x <2.0.0'` as your dependency constraint to allow a smoother ecosystem migration.
For more details see: https://github.com/flutter/flutter/wiki/Package-migration-to-1.0.0
